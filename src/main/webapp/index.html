Crafty-modules is a package manager for the Crafty game engine. It is the preferred way to share, find and use components and other Crafty extensions.

Using a module in your game

Crafty.modules({ moveto: 'DEV' }, function () {
    //module is ready
    Crafty.e("MoveTo, 2D, DOM");
});

The module is downloaded asynchroneously at runtime.
